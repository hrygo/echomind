# 📅 产品路线图 - EchoMind

**当前阶段**：Genesis (产品诞生)
**发布周期**：双周迭代 (Bi-weekly Sprint)

---

## 🗓️ Phase 1: 基础连接与数据概览 (MVP)
**目标**：跑通 "连接 -> 同步 -> 展示" 核心链路，验证技术可行性。
**预计周期**：4 - 6 周

*   **后端 (Backend)**
    *   [ ] 实现通用 IMAP 连接器 (Go)，支持 SSL/TLS 连接。
    *   [ ] 实现邮件元数据同步 (Subject, Sender, Date) 存入 PostgreSQL。
    *   [ ] 基础邮件正文解析 (HTML -> Text)。
*   **前端 (Frontend)**
    *   [ ] 登录/注册页 & 邮箱绑定向导 (Add Account Wizard)。
    *   [ ] 统一收件箱列表视图 (Unified Inbox)。
    *   [ ] 邮件详情页 (基础阅读体验)。
*   **AI (Prelim)**
    *   [ ] 接入 DeepSeek API。
    *   [ ] 实现简单的 "一键摘要" 功能 (On-demand summary)。

## 🗓️ Phase 2: 智能分析与双视图构建 (Alpha)
**目标**：实现核心价值功能，区分高管/管理者视图。
**预计周期**：6 - 8 周

*   **核心功能**
    *   [ ] **智能分类器**：自动区分 Newsletter, Notification, Personal, Work。
    *   [ ] **每日日报 (Daily Digest)**：每天早上 8:00 生成简报邮件/推送。
    *   [ ] **任务提取引擎**：从邮件中识别 Action Items 并存入待办库。
    *   [ ] **微信公众号接入 (Beta)**：
        *   [ ] 实现微信服务器验签与基础消息回复。
        *   [ ] 实现账户绑定流程 (OpenID <-> UserID)。
        *   [ ] 上线 "早报推送" 模板消息。
*   **用户界面**
    *   [ ] **高管仪表盘**：开发 "决策概览" 页面，隐藏琐碎邮件列表。
    *   [ ] **管理者工作台**：开发 "任务看板" (Kanban/List)，集成邮件处理。
*   **技术基建**
    *   [ ] 引入消息队列 (Redis/Kafka) 异步处理 AI 任务。
    *   [ ] PII 脱敏过滤器上线。

## 🗓️ Phase 3: 关系网络与深度整合 (Beta)
**目标**：增强粘性，提供深层次洞察，拓宽用户群体至销售/投资人。
**预计周期**：8 - 10 周

*   **高级功能**
    *   [ ] **人脉价值图谱**：可视化展示联系人网络及交互热度（与关系热度图融合）。
    *   [ ] **意图雷达 (Intent Radar)**：从邮件中自动识别购买/合作/会议等高价值意图。
    *   [ ] **关系热度图 (Relationship Heatmap)**：前端实现，可视化联系人互动频率及冷暖。
    *   [ ] **时间线旅程 (Timeline)**：以联系人为中心，聚合所有相关邮件、会议和附件。
    *   [ ] **智能草稿 (Smart Draft)**：基于意图自动生成回复。
    *   [ ] **情绪雷达**：识别并高亮 "愤怒/紧急" 的邮件。
    *   [ ] **微信指令交互**：支持回复 "批准/拒绝" 直接处理待办。
*   **生态集成**
    *   [ ] 日历集成 (CalDAV)：从邮件直接创建会议日程。
    *   [ ] 移动端适配 (PWA / Mobile Web)。

## 🗓️ Phase 4: 商业化与规模化 (Public Launch)
**目标**：公开上线，开始商业验证。
**预计周期**：持续演进

*   **商业化**
    *   [ ] Stripe 支付集成。
    *   [ ] 套餐权限管理 (Free/Pro/Team)。
*   **企业级特性**
    *   [ ] 团队协作：分享邮件/Thread 给同事，添加内部评论。
    *   [ ] SOC2 合规审计准备。
    *   [ ] 私有化部署支持 (Docker Compose/Helm)。
