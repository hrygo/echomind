# EchoMind Documentation

This directory contains all the documentation for the EchoMind project.

## Structure

- **[product/](./product/)**: Product Requirements Documents (PRD), roadmaps, design documents, and implementation plans.
- **[architecture/](./architecture/)**: Technical architecture, framework details, decision records (ADRs), and specific technical guides (e.g., Vector Search, OTel).
- **[api/](./api/)**: API specifications (OpenAPI), sequence diagrams, and usage examples.
- **[releases/](./releases/)**: Release notes, iteration summaries, sprint planning, and completion reports.

- **Product Roadmap**: [product/product-roadmap.md](./product/product-roadmap.md)
- **Architecture Overview**: [architecture/architecture.md](./architecture/architecture.md)
- **API Specification**: [api/openapi.yaml](./api/openapi.yaml)
